# Gulf-India Crisis Resource Hub

Emergency information aggregator for Indian nationals affected by the March 2026 Gulf crisis.

## Features

- **Embassy Emergency Helplines**: 24/7 numbers for Indian embassies in Dubai, Abu Dhabi, Tehran, Doha, and Muscat
- **CBSE Exam Updates**: Official circular and affected country list for March 2 postponements
- **Airline Rebooking**: Direct flight status and refund links for Air India, IndiGo, and Emirates
- **Airport Status**: Live status for DXB and DOH
- **MADAD Portal**: Quick link to MEA consular services registration

## Architecture

- Static HTML/CSS/JS — no build step required
- Data-driven via `data.json` — update the JSON to update the site
- Mobile-first, responsive design with dark/light mode
- GoatCounter analytics (privacy-friendly, no cookies)
- All phone numbers are tappable `tel:` links
- All WhatsApp numbers link to `wa.me/`
