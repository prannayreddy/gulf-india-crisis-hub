# Gulf-India Crisis Resource Hub

**Live Site: [gulf-india-crisis-hub.vercel.app](https://gulf-india-crisis-hub.vercel.app/)**

Emergency resource hub for Indian nationals affected by the March 2026 Gulf crisis (Iran-Israel conflict). Provides real-time information on:

- Embassy & Consulate emergency helplines (UAE, Iran, Qatar, Oman, Saudi Arabia, Kuwait, Bahrain)
- MEA 24/7 Control Room contacts
- Flight disruption status for 12+ airlines
- Airport operations status (7 Gulf airports)
- CBSE exam postponement details
- Live updates feed with verified sources

## Features

- Auto-monitored and updated regularly
- Mobile-first responsive design with dark/light mode
- One-tap calling for all emergency numbers
- WhatsApp integration for embassy contacts
- Clean data architecture — all content driven from `data.json`

## Data Sources

All information is aggregated from official sources including:
- Indian Embassy/Consulate official websites
- CBSE official circulars
- MEA MADAD Portal
- Airline official portals
- News sources (verified only)
